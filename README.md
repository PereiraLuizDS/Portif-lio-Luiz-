# Conheça-me 
Meu nome é Luiz Otávio. Sou estudante de Ciência de Dados e Técnico em Administração,
com interesse em tecnologia, análise de dados e desenvolvimento de soluções baseadas
em informação. Possuo experiência nas áreas administrativa e logística, onde atuei
com controle de estoque, organização de processos e utilização de sistemas ERP/WMS.

Atualmente busco aprimorar minhas competências técnicas e analíticas na área de
dados, desenvolvendo projetos e adquirindo habilidades em ferramentas e linguagens
como Python e SQL para análise, manipulação e interpretação de dados.

Meu objetivo é crescer profissionalmente na área de tecnologia e ciência de dados,
utilizando programação e análise de dados para transformar informações em
conhecimento útil para a tomada de decisões.
